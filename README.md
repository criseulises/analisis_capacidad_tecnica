# ANÁLISIS CONSOLIDADO DEL EQUIPO DE DESARROLLO

## Archivos Generados

Este paquete contiene 3 archivos principales:

### 1. Analisis_Completo_Equipo_Desarrollo.docx
**Documento Word con análisis exhaustivo**
- Resumen ejecutivo consolidado al inicio
- Análisis detallado de cada uno de los 8 perfiles
- Evaluación de fortalezas, debilidades y oportunidades
- Análisis completo de tecnologías del equipo
- Evaluación de oportunidad: Automatización de IA con n8n y ElevenLabs
- Análisis completo del sector bancario
- Recomendaciones estratégicas finales

### 2. index.html
**Página web interactiva con análisis general y gráficos**

**Contenido:**
- Resumen ejecutivo
- Estadísticas del equipo
- Gráficos interactivos:
  - Distribución de experiencia
  - Tecnologías backend
  - Tecnologías frontend/mobile
  - Bases de datos
  - Análisis de viabilidad para proyecto de IA
  - Capacidades bancarias
- Análisis de oportunidad: Automatización de IA
- Análisis de oportunidades en sector bancario
- Recomendaciones estratégicas

**Cómo usar:**
1. Abrir el archivo index.html en cualquier navegador web
2. Navegar usando el menú superior
3. Los gráficos son interactivos (hover para ver detalles)
4. Hacer clic en "Ver Perfiles Individuales" para ir a perfiles.html

### 3. perfiles.html
**Página web con perfiles individuales detallados**

**Contenido:**
- Cards interactivas para cada uno de los 8 desarrolladores
- Al hacer clic en cada card, se abre un modal con:
  - Información de contacto
  - Educación completa
  - Experiencia laboral detallada
  - Stack tecnológico completo
  - Fortalezas
  - Debilidades/Oportunidades de mejora
  - Alineación con oportunidades de negocio

**Cómo usar:**
1. Abrir el archivo perfiles.html en cualquier navegador web
2. Hacer clic en cualquier tarjeta de perfil para ver detalles completos
3. Cerrar el modal haciendo clic en la X o fuera del modal
4. Usar el botón "Volver al Análisis General" para regresar a index.html
